1 栈基础 / Stack Basics (LIFO)

🥞

类比 / Analogy

栈就像一摞煎饼：你只能从最上面放（push）和拿（pop）。最后放上去的煎饼最先被拿走——这就是 LIFO（Last In, First Out，后进先出）。

Push 放到栈顶，Pop 从栈顶取出，都是 O(1)

2 队列基础 / Queue Basics (FIFO)

🚶‍♂️

类比 / Analogy

队列就像排队买奶茶：先来的人先买到（先进先出）。新来的人排到队尾（enqueue），买完的人从队头离开（dequeue）。

3 括号匹配 / Bracket Matching

左括号入栈，右括号与栈顶匹配则弹出，最终栈空 = 合法
// 括号匹配 / Valid parentheses
boolean isValid(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    Map<Character, Character> map = Map.of(')', '(', '}', '{', ']', '[');
    for (char c : s.toCharArray()) {
        if (map.containsValue(c)) stack.push(c);
        else if (stack.isEmpty() || stack.pop() != map.get(c)) return false;
    }
    return stack.isEmpty();
}

4 单调栈 / Monotonic Stack

📐 什么是单调栈？

栈内元素始终保持单调递增或递减。新元素入栈前，弹出所有破坏单调性的元素。经典应用：下一个更大元素（Next Greater Element）。

单调栈保证每个元素只被处理一次，总时间 O(n)
// 下一个更大元素 / Next greater element
int[] nextGreater(int[] nums) {
    int n = nums.length;
    int[] res = new int[n];
    Arrays.fill(res, -1);
    Deque<Integer> stack = new ArrayDeque<>(); // 存索引
    for (int i = 0; i < n; i++) {
        while (!stack.isEmpty() && nums[stack.peek()] < nums[i]) {
            res[stack.pop()] = nums[i];
        }
        stack.push(i);
    }
    return res;
}

5 最小栈 / Min Stack

两个栈同步操作，最小栈栈顶始终是当前最小值
// 最小栈 / Min Stack - O(1) push, pop, getMin
class MinStack {
    Deque<Integer> stack = new ArrayDeque<>();
    Deque<Integer> minStack = new ArrayDeque<>();

    void push(int val) {
        stack.push(val);
        minStack.push(minStack.isEmpty() ? val : Math.min(val, minStack.peek()));
    }
    void pop() { stack.pop(); minStack.pop(); }
    int top() { return stack.peek(); }
    int getMin() { return minStack.peek(); }
}

6 用两个栈实现队列 / Queue Using Two Stacks

倒入后顺序反转，实现 FIFO。均摊 O(1)
// 用两个栈实现队列 / Queue using two stacks
class MyQueue {
    Deque<Integer> in = new ArrayDeque<>(), out = new ArrayDeque<>();

    void push(int x) { in.push(x); }

    int pop() { transfer(); return out.pop(); }

    int peek() { transfer(); return out.peek(); }

    private void transfer() {
        if (out.isEmpty()) while (!in.isEmpty()) out.push(in.pop());
    }
}

🎯 栈和队列的选择
  • 需要"最近"匹配（括号、标签）→ 栈
  • 需要"先来先服务"（BFS、任务调度）→ 队列
  • 需要"下一个更大/更小"→ 单调栈
  • 需要 O(1) 获取最值→ 辅助栈/单调队列
