1 Scala 语言特性

1.1 OOP + FP

  • 面向对象：一切皆对象，`1 + 1` 实际是 `1.+(1)`
  • 函数式：函数为头等公民，方法无副作用
  • 特质（trait）：比接口更强大

1.2 并发特性

3 LSM Tree

3.1 概念

Log-Structured Merge Tree，结合 Immutable 思想实现的高效存储结构。

3.2 优势

  • 写性能高（顺序写）
  • 支持高并发写入
  • 适用于写多读少场景

3.3 应用

RocksDB
Cassandra
LevelDB

4 Doris Colocate Join

4.1 概念

将关联数据存储在同一批次节点，避免跨节点网络开销。

4.2 核心思路

  • 导入时保证数据本地性
  • 查询时利用本地性优化
