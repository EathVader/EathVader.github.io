1 基础二分 / Basic Binary Search

📖

类比 / Analogy

在一本按页码排列的字典中找某个词：翻到中间，如果目标在前半部分就往前翻，在后半部分就往后翻。每次翻一半，几次就能找到。这就是二分搜索——每次排除一半的搜索空间。

每轮排除一半，3 轮找到目标（log₂7 ≈ 3）
// 基础二分搜索 / Basic binary search
int binarySearch(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;  // 防溢出
        if (nums[mid] == target) return mid;
        else if (nums[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}
⚠️ 常见 Bug

mid = (left + right) / 2 当 left + right 超过 int 最大值时会溢出！用 left + (right - left) / 2 更安全。

2 找第一个/最后一个位置 / Find First & Last Position

🎯

类比 / Analogy

在一排相同颜色的球中找最左边和最右边的那个。找到一个后不能停，要继续缩小范围，直到确认是边界。

// 找第一个等于 target 的位置
int findFirst(int[] nums, int target) {
    int left = 0, right = nums.length - 1, ans = -1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] >= target) { if (nums[mid] == target) ans = mid; right = mid - 1; }
        else left = mid + 1;
    }
    return ans;
}

// 找最后一个等于 target 的位置
int findLast(int[] nums, int target) {
    int left = 0, right = nums.length - 1, ans = -1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] <= target) { if (nums[mid] == target) ans = mid; left = mid + 1; }
        else right = mid - 1;
    }
    return ans;
}
💡 关键区别

找到 target 后：找第一个→ 继续往左搜（right = mid - 1）；找最后一个→ 继续往右搜（left = mid + 1）。

3 旋转数组搜索 / Search in Rotated Sorted Array

旋转后数组分成两段有序部分，mid 一定落在其中一段
// 旋转数组搜索 / Search in rotated sorted array
int search(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) return mid;
        if (nums[left] <= nums[mid]) {  // 左半有序
            if (nums[left] <= target && target < nums[mid]) right = mid - 1;
            else left = mid + 1;
        } else {  // 右半有序
            if (nums[mid] < target && target <= nums[right]) left = mid + 1;
            else right = mid - 1;
        }
    }
    return -1;
}
💡 核心思路

每次二分后，至少有一半是有序的。判断 target 是否在有序的那一半范围内，如果是就搜那半，否则搜另一半。

4 寻找峰值 / Find Peak Element

如果 mid 的右邻居更大，峰值一定在右边；反之在左边
// 寻找峰值 / Find peak element - O(log n)
int findPeakElement(int[] nums) {
    int left = 0, right = nums.length - 1;
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] < nums[mid + 1]) left = mid + 1;  // 上坡，峰在右
        else right = mid;  // 下坡或峰顶，峰在左（含 mid）
    }
    return left;
}

5 搜索二维矩阵 / Search 2D Matrix

将二维坐标映射为一维索引，直接二分
// 搜索二维矩阵 / Search 2D matrix
boolean searchMatrix(int[][] matrix, int target) {
    int m = matrix.length, n = matrix[0].length;
    int left = 0, right = m * n - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        int val = matrix[mid / n][mid % n];  // 一维→二维
        if (val == target) return true;
        else if (val < target) left = mid + 1;
        else right = mid - 1;
    }
    return false;
}

7 总结 / Summary

问题核心思路时间
基础二分left ≤ right，每次排除一半O(log n)
找第一个/最后一个找到后继续缩小范围O(log n)
旋转数组至少一半有序，判断 target 在哪半O(log n)
寻找峰值上坡搜右，下坡搜左O(log n)
搜索二维矩阵一维索引映射二维坐标O(log(mn))
🎯 二分法使用条件
  • 搜索空间具有单调性或可以二段性划分
  • 每次可以排除一半的搜索空间
  • 不一定要求数组严格有序——只要能判断"答案在左还是右"
